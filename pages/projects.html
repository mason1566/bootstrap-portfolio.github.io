Project Name: Guitar Helper

Status: Finished

Guitar Helper is a simple command-line guitar program that helps the user learn and practice their recognition of intervals between notes.

The user is presented with an ascii representation of the fretboard that has a single fret marked 'I', and another marked 'x'.

The user is then tasked to try to identify the given interval and answer in interval form: 1 - b2 - 2 - b3 - 3 - 4 - b5 - 5 - b6 - 6 - b7 - 7.

If you don't know anything about intervals, I recommend going here.

Outcome:

I feel like a learned a lot from this tiny project. It was my first time using classes (OOP) and it really helped me to solidify the concept. I'm really pleased with how it works and have many cool features planned for the future.

Project Name: Knowledge Base

Status: Ongoing

I first got the inspiration to do this after watching a video about a guy called Niklas Luhmann. He wrote over 70 books in his life and published over 400 scholarly articles; and he attributed it all to his note-taking system.

He prioritized making links and forming connections between notes. If you would like to read more about this method of note-taking, check out zettelkaste.de.

The app I use, called Obsidian, has a fantastic way of linking notes together. It allows you to seamlessly add wikipedia style links anywhere by simply typing: [[Put the name of your note inside these brackets]].

Experience:

It has been a very hard process to get familiar with. I discovered it a few weeks before our semester started, going as far as reading a book about the specific note-taking style.

I still have a lot of learning to do, but hopefully I am setting myself up for the future having a system like this in place.

The coolest thing is that my notes sync up across all my devices (It's $7.99 a month, but there are ways to do it freely if needed).
